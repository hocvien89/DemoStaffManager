-------------------
Cài đặt vue: 
	npm install vue
Cài đặt vue-cli: 
	npm install vue-cli 
hoặc cài đặt toàn cục: 
	npm install --global vue-cli
Khởi tạo một project vue: 
	vue init webpack my-project   (với my-project là tên project cần tạo)
Chạy thử project sau khi khởi tạo gõ: 
	cd my_project
	npm run dev
Vào localhost kiểm tra, sẽ thấy màn hình mặc định khi khởi tạo của vue

-------------------
Trong quá trình phát triển project, cài thêm thư viện theo cú pháp:
	npm install lib-name       (với lib-name là tên thư viện cần cài đặt)
-------------------
Sau khi clone source code từ github về, chạy câu lệnh sau để cài đặt các node_modules:
	npm install
Để chạy project gõ tiếp:
	npm run dev
Vào localhost xem kết quả.
